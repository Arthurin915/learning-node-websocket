Para executar o servidor, digite: " npm start " no terminal no nível root do projeto
Para executar o cliente, abra o arquivo cliente1.html no browser ( ele automaticamente se conectará ao websocket)

Ao abrir o cliente, abra o DevTools do seu browser e vá para a aba Console, as mensagens retornadas pelo servidor aparecerão lá

De preferencia abra no google chrome, não foi testado em outros browsers
